Apa Itu RAG?

Retrieval Augmented Generation

RAG menggabungkan pencarian informasi dengan generasi teks. AI mengakses sumber data eksternal langsung sebelum menjawab.

AI tidak menjawab dari ingatannya sendiri, tapi mencari fakta dulu di dokumen kamu, baru menjawab.

Vector Database

Cara Lama

  • Pecah dokumen jadi chunk kecil
  • Convert ke vector embedding
  • Simpan di database terpisah
  • Cari berdasarkan similarity
  • Butuh infra tambahan
3 KOMPONEN

PageIndex

Cara Baru

  • Pertahankan struktur dokumen utuh
  • Bangun tree hierarchy
  • Reasoning langsung di cloud
  • Cari berdasarkan semantic
  • Semua handled by PageIndex
