Sesi 5 — 10 Menit

Struktur Project

Memahami bagaimana file-file dalam project disusun

project/ ├── agents/ ← INTI: Agent code │ └── openai-agents-starter/ │ ├── index.ts ← Main logic agent │ └── stop.ts ← Stop handler ├── static/ ← Frontend │ ├── index.html │ ├── style.css │ └── app.js ├── edgeone.json ← Config project └── package.json

agents/

Semua file di folder ini menjadi POST endpoint. Contoh: agents/chat/index.ts menjadi POST /chat. Setiap request memerlukan conversation_id.

static/

File frontend yang di-serve secara otomatis. index.html adalah halaman chat yang tampil di browser.

edgeone.json

Konfigurasi project: framework, model AI, environment variables, dan pengaturan deploy.

Sesi 6 — 20 Menit Praktik

Mengubah Instruksi Agent

Membuat agent menjadi lebih personal dan sesuai kebutuhan

Buka file agents/chat/index.ts, temukan bagian berikut:

Sebelum
const agent = new Agent({ name: 'Assistant', instructions: 'Kamu adalah asisten...', model: '@makers/deepseek-v4-flash', });
Sesudah
const agent = new Agent({ name: 'Mentor KF', instructions: `Kamu adalah mentor coding KelasFullstack. Panggil user "kawan" atau "bro". Jawab dengan Bahasa Indonesia santai. Kasih contoh kode kalau perlu. Jangan formal.`, model: '@makers/deepseek-v4-flash', });
!

Setelah mengubah kode, klik tombol "Deploy" / "Redeploy" di dashboard.

Tunggu 20-30 detik, lalu refresh URL project dan coba chat kembali.

Sesi 6 — Lanjutan Praktik

Menambahkan Function Tool

Memberikan kemampuan baru kepada agent — contoh: kalkulator BMI

Tambahkan kode berikut di file agents/_tools.ts:

export const hitungBMI = { name: "hitung_bmi", description: "Hitung BMI dari berat dan tinggi badan", parameters: { type: "object", properties: { berat: { type: "number", desc: "Berat dalam kg" }, tinggi: { type: "number", desc: "Tinggi dalam cm" }, }, required: ["berat", "tinggi"], }, handler: async ({ berat, tinggi }) => { const bmi = berat / ((tinggi / 100) ** 2); let kategori = ""; if (bmi < 18.5) kategori = "Kurus"; else if (bmi < 25) kategori = "Normal"; else if (bmi < 30) kategori = "Gemuk"; else kategori = "Obesitas"; return `BMI: ${bmi.toFixed(1)} — ${kategori}`; }, };
Redeploy project
Chat: "Hitung BMI saya 70kg 170cm"
Agent menjawab dengan hasil perhitungan!
Sesi 7 — 10 Menit Praktik

Mengganti Model AI

Salah satu keunggulan EdgeOne: ganti model cukup ubah 1 parameter

Di agents/chat/index.ts, ubah baris berikut:

model: '@makers/deepseek-v4-flash',

Ganti ke salah satu model di bawah ini:

@makers/deepseek-v4-flash

To the point, teknis

@makers/hy3-preview

Lebih panjang, detail

@makers/minimax-m2.7

Santai, simpel

Tantangan: Bandingkan Jawaban

Ajukan pertanyaan yang sama ke 3 model berbeda: "Jelaskan konsep REST API dalam 3 kalimat". Perhatikan perbedaan gaya jawabannya.

Model apapun, API Key tetap sama. Inilah kekuatan AI Gateway!
