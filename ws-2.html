Sesi 1 — Lanjutan

Apa yang Akan Dibuat?

Membangun Enterprise RAG Agent dari awal

Enterprise RAG Agent

Upload PDF AI Membaca Tanya Apa Saja Jawab + Citation

Apa itu RAG?

Retrieval Augmented Generation — agent dapat membaca file yang diupload, lalu menjawab pertanyaan berdasarkan isi file tersebut. Cocok untuk HR, legal, edukasi, dan customer service.

Mengapa Ini Berbeda?

1 Domain

Frontend dan backend di satu alamat

1 Project

Web dan Agent dalam satu repositori

No CORS Drama

Tidak ada masalah cross-origin

"Sesi 2 ini yang membuat EdgeOne Makers berbeda dari platform lain. Web (frontend) + AI Agent (backend) = SATU project. Satu domain. Satu deploy. Tanpa konfigurasi CORS yang rumit."

Sesi 6 — 15 Menit

Blob Storage

Di mana file yang diupload disimpan?

User Upload PDF
EdgeOne terima (multipart/form-data)
Blob Storage: documents/{conversation_id}/{filename}
getStore("documents").get("path/to/file.pdf")

Frontend Upload

async function uploadFile(file, convId) { const fd = new FormData(); fd.append('file', file); fd.append('conversation_id', convId); const res = await fetch('/api/upload', { method: 'POST', body: fd, }); return res.json(); }

Backend Handler

import { getStore } from "@tencent/pages-blob"; export default async function handler(req) { const fd = await req.formData(); const file = fd.get('file'); const convId = fd.get('conversation_id'); const store = getStore("documents"); const path = `${convId}/${file.name}`; await store.set(path, await file.arrayBuffer() ); return Response.json({ success: true, path, filename: file.name }); }
Sesi 6 — Lanjutan

Blob Storage vs KV Storage

Aspek Blob KV
Tipe DataFile / binary / objectKey-value (string)
Max Size25 MB per object25 MB per value
StrukturDirectory-based (hierarki)Flat namespace
Use CaseDokumen, gambar, fileConfig, counters, session

Untuk RAG Agent, kita menggunakan Blob Storage karena menyimpan file PDF yang memiliki struktur hierarki berdasarkan conversation ID.

Sesi 7 — 10 Menit

Prompt Engineering untuk RAG

Mengatur system prompt agar jawaban agent lebih akurat

const agent = new Agent({ name: "DocumentAssistant", instructions: `Kamu adalah asisten dokumen KelasFullstack. ATURAN MAIN: 1. Jawab berdasarkan dokumen yang diupload — JANGAN mengarang 2. Kalau ditanya di luar dokumen, bilang "Maaf, info itu tidak ada di dokumen" 3. Sertakan nomor halaman kalau bisa 4. Jawab menggunakan Bahasa Indonesia santai 5. Kalau belum upload dokumen, arahkan user untuk upload dulu POLA JAWABAN: - Ringkasan: "Berdasarkan dokumen, poin utamanya adalah..." - Detail: "Di halaman X disebutkan bahwa..." - Luar dokumen: "Maaf, dokumen tidak mencakup informasi itu."``, });

Tugas — 5 Menit

Tambahkan dua aturan berikut ke dalam instruksi agent:

  • Jika user bertanya dalam Bahasa Inggris, jawab dalam Bahasa Inggris
  • Jika jawaban memerlukan langkah-langkah, gunakan numbering

Sesi 9 — 10 Menit

Catatan Produksi

Bring Your Own Key (BYOK)

Built-in model hanya untuk testing (500K token gratis). Untuk produksi, gunakan API Key sendiri melalui Settings → Environment Variables → tambah OPENAI_API_KEY

Custom Domain

Ingin menggunakan domain sendiri? Melalui Settings → Domain → Add Custom Domain. SSL otomatis dari EdgeOne, tanpa biaya tambahan.

Monitoring

Dashboard → Metrics untuk melihat jumlah request, latency, dan error rate. Trace viewer untuk melihat LLM calls, tool calls, dan durasi setiap proses.

Keamanan

Jangan hardcode API Key di kode — gunakan context.env

Lakukan input validation di frontend dan backend

EdgeOne secara otomatis menangani DDoS protection
