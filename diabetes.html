Fundamental Question

Pernahkah Terpikir?

"Mengapa kita harus membayar mahal untuk token AI (LLM), ketika ada mesin pembelajaran (Machine Learning) klasik yang gratis, cepat, dan akurat untuk kasus spesifik seperti diagnosa medis?"

Tantangan LLM / AI Token

  • 01
    Biaya Tak Terduga

    Setiap prediksi memakan token. Semakin banyak pasien, semakin besar biaya yang harus dikeluarkan.

  • 02
    Latency & Koneksi

    Membutuhkan internet stabil. Bagaimana jika rumah sakit sedang offline?

  • 03
    Privasi Data

    Data pasien dikirim ke server pihak ketiga. Apakah ini sesuai standar etika medis?

Solusi: Pure Machine Learning

  • 01
    Investasi Sekali, Untung Selamanya

    Tanpa biaya berlangganan token. Model milik kita sepenuhnya.

  • 02
    Kecepatan Milidetik

    Prediksi instan di perangkat lokal. Tidak ada lag, tidak ada tunggu.

  • 03
    Keamanan Maksimal

    Data pasien tidak keluar dari server rumah sakit. Privasi terjaga 100%.

Basic Python for Data Science

The Swiss Army Knife: Pandas

Mengapa Pandas Itu Wajib?

Jika Python adalah bahasa, maka Pandas adalah tata bahasanya. Hampir 90% waktu seorang Data Scientist dihabiskan di sini—membersihkan, memanipulasi, dan memahami data.

"Data di dunia nyata itu kacau. Pandas adalah sapu yang merapikannya."

Operasi Kunci:

  • Read/Write: Membaca CSV, Excel, SQL.
  • Slicing: Memilih data spesifik yang dibutuhkan.
  • Cleaning: Menghapus data kosong atau duplikat.
  • Aggregation: Menghitung rata-rata, total, atau kelompok data.

Contoh Praktis Nyata

import pandas as pd

# 1. Load data (Langkah pertama)
df = pd.read_csv('diabetes.csv')

# 2. Cek kesehatan data
df.info() # Apakah ada data kosong?
df.describe() # Bagaimana statistiknya?

# 3. Filter data spesifik
high_risk = df[df['Age'] > 50]
Insight: Menguasai Pandas berarti menguasai 50% pekerjaan Data Science. Tanpa ini, model tidak akan pernah bisa dibangun.

09

Model Evaluation

Beyond Accuracy

Scenario: Model memiliki akurasi 90%. Tapi... itu berarti 10% pasien salah diagnosa. Dalam konteks medis, apakah itu bisa diterima?

Metrik yang Benar-Benar Penting

Recall (Sensitivity)

Dari semua orang yang SAKIT, berapa persen yang berhasil kita deteksi? (Ini paling penting!)

Precision

Dari semua prediksi "Positif", berapa yang benar-benar sakit?

F1-Score

Keseimbangan antara Precision dan Recall.

Confusion Matrix: The Truth Teller

TP
Pasien positif, diprediksi positif
FP
Pasien sehat, diprediksi positif (False Alarm)
FN
Pasien positif, diprediksi sehat (BERBAHAYA!)
TN
Pasien sehat, diprediksi sehat

Tujuan kita: Minimalisir False Negative (FN) sekecil mungkin.

Fine-Tuning The Engine

Dari "Cukup" ke "Sempurna"

GridSearchCV

Mencoba setiap kombinasi secara sistematis. Teliti tapi lambat.

RandomSearchCV

Implementation Code

# Definisikan "ruang" pencarian
params = {
  'n_estimators': [50, 100, 200],
  'max_depth': [10, 20, None]
}

# Mulai pencarian
